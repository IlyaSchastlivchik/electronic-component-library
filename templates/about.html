О проекте

AI Component Library

Открытая база электронных компонентов с ИИ-ассистентом

Цель проекта

Создать открытую, машиночитаемую базу данных электронных компонентов с возможностью интеллектуального поиска через естественный язык.

Решаемые проблемы:
  • Отсутствие единой базы ВАХ в цифровом формате
  • Сложность поиска компонентов по параметрам
  • Необходимость ручного перебора datasheet
  • Отсутствие открытых API для доступа к данным компонентов
Возможности
  • Поиск на естественном языке - задавайте вопросы как человеку
  • Фильтрация по параметрам - ток, напряжение, мощность, тип
  • Визуализация ВАХ - интерактивные графики характеристик
  • Открытый API - REST API для разработчиков
  • Коллективное развитие - база пополняется сообществом
  • Кроссплатформенность - работает на Windows, Linux, macOS
